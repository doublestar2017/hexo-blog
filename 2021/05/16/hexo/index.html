搭博客吗？十分钟的那种

发表于 2021-05-16 更新于 2021-05-25 分类于 博客

宗旨

几行命令搭建博客，争 10min 保 1h

注：hexo，中文：十六进制(我不会读 -_-)

一、开始

1.安装 Hexo

npm install -g hexo-cli

2.验证是否安装成功

hexo -v

3.初始化文件夹

hexo init <文件夹名称>

4.安装依赖

npm install

5.运行

npm run server / hexo s

6.创建新文章

hexo new "test"

7.安装 hexo 的 git 扩展

npm i hexo-deployer-git

8.配置 git

_config.yml 文件

deploy:
type: git
repository: https://github.com/doublestar2017/hexo-blog.git
branch: master

如果您的网站存放在子目录中，例如 http://example.com/blog，则请将您的 url 设为 http://example.com/blog 并把 root 设为 /blog/。

8.生成静态网页

hexo g

9.提交

hexo d

10.更换主题(xxxx 是主题名)

git clone github地址 themes/xxxx

_config.yml 文件,注释掉原来的 theme 并新增一句 theme: xxxx
如果 clone 过程过于缓慢，可以直接下载主题代码包到 themes 文件夹下
